There are 2 entry points to the program

 Crawler	This class is the entry point for the WebCrawler program to handle single json files.
 		The program requires the full path(including json file) to be passed.
 		
 CrawlThread	This class is implemented as a threaded approach for the WebCrawler program to handle multiple json files.
		This takes the full path to the directory, where the json files exist, to be passed.
